В проекте под Table подразумевается двухмерный массив где первый индекс это x, а второй y те [x,y].
Вся индексация в проекте происходит с левого верхнего угла (0,0) в право для x и в низ для y.
Файл для испорорта данных ".../Assets/Resources/input.txt"
Входные данные должны делиться на строчки. Все строчки должны быть одинакого размера, без пробелов и других знаков,
которых нету в славоре.
В проекте есть Словарь с char на ShapeData  ".../Assets/Configs/TextToShapeDictionaryDictionary.asset", 
в нем можно прописывать новые фигуры или добовлять дополнительные ассоциации с символами. Проверки на повторяющие char не происходит.
те чтобы добавить новую фигуру (цвет) просто создайте новый ShapeData и добавьте ассоциацию в словарь.
